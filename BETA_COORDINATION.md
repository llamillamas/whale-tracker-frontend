## Beta Testing Guide

### Getting Started

#### 1. Account Setup
1. Visit https://whale-tracker.vercel.app
2. Click "Connect Wallet" (MetaMask)
3. Sign the message to verify ownership
4. Set username (unique identifier)

#### 2. Explore the Leaderboard
1. Go to "Whales" page
2. Sort by ROI, Trades, Followers
3. Click on whale profile
4. View trade history

#### 3. Copy Your First Whale
1. Find a whale you want to follow
2. Click "Copy Trader" button
3. Select 3-5 recent trades
4. Choose 25% copy allocation
5. Click "Confirm & Copy"

#### 4. Monitor Your Copy
1. Go to Dashboard
2. Click "Copied Traders" tab
3. See performance vs whale
4. Pause or stop anytime

## Bug Reporting Guide

### How to Report a Bug
1. **Describe Issue:** What happened?
2. **Steps to Reproduce:** How did it happen?
3. **Expected Result:** What should happen?
4. **Actual Result:** What actually happened?
5. **Screenshot/Video:** Evidence (if possible)

### Bug Report Template
```
**Title:** [COMPONENT] Brief description

**Description:**

I see...

**Screenshot:**
```

## FAQ

**Q: What if I find a bug?**
A: Report it on GitHub or Discord. We'll fix it and send you a bonus!

**Q: Can I stop copying anytime?**
A: Yes! Click "Stop" on any copy trade to cancel immediately.

**Q: Is my data safe?**
A: 100% secure. We don't hold your keys or funds.

**Q: What if the whale makes losing trades?**
A: You copy them too. Risk is YOUR responsibility.

**Q: How often are whales updated?**
A: Real-time! Copy trades execute within 1 second.

**Q: Can I copy multiple whales?**
A: Yes! Copy as many as you want.

**Q: Is there a refund period?**
A: This is beta. No refunds, but you can stop anytime.
